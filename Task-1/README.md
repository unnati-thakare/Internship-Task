## Description
This project is a simple command-line based To-Do List application
developed using Python. It helps users manage their daily tasks.

## Logic Used
- A list is used to store tasks
- Each task is stored as a dictionary with task name and status
- Menu-driven approach
- While loop keeps the program running until exit

## Features
1. Add task
2. View tasks
3. Mark task as completed
4. Delete task
5. Exit

## How to Run
1. Open terminal in the project folder
2. Run: python todo.py
